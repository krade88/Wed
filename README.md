# Wedding site (static)

Автономный статический сайт‑приглашение (HTML/CSS/Vanilla JS).

## Запуск

### Вариант 1 (самый простой)

Откройте файл `wedding-site/index.html` в браузере.

### Вариант 2 (локальный сервер)

Если у вас установлен Python:

```bash
cd wedding-site
python -m http.server 5173
```

Откройте `http://localhost:5173`.

## Технические детали

- Шрифты: Google Fonts `Great Vibes` + `Montserrat` (free, OFL).
- Таймер: до **20.06.2026 16:00 (МСК, UTC+3)**.
- Форма: валидация на клиенте, сохранение черновика в `localStorage`, отправка в mock endpoint.

## Ассеты / лицензии

- Фон интро (`img/bg/intro.png`) взят из предоставленного вами скриншота референса (используется только для воссоздания макета).
